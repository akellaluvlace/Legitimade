# LegitimaDe App - Setup Instructions

## Prerequisites
- Node.js 16+ installed
- Expo CLI installed (`npm install -g expo-cli`)
- Expo Go app on testing device

## Installation
1. Clone repository
2. Run `npm install`
3. Run `expo start`
4. Scan QR code with Expo Go app

## Configuration
- Update SOLANA_TX_URL in src/constants/config.ts
- Replace placeholder logo in assets folder

## Testing
- Any QR code will show "Product is Genuine"
- Track Journey button opens Solana explorer
